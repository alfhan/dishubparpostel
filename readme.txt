Di butuhkan bantuan untuk modifikasi aplikasi berbasis laravel 4.2 yang sudah ada untuk perbaikan UI dan beberapa fungsi yang belum jalan. Penambahan homepage, about us, contact page yang bisa di edit dari admin page, dengan editor "wyswyg". Untuk area admin, di butuhkan pembuatan role user, level admin dan operator. operator hanya bisa input data saja sedangkan admin bisa edit data dan mengubah beberapa parameter terkait dengan penambahan user dan beberapa setting penting aplikasi. Kemudian ada beberapa fungsi yang perlu di perbaiki dalam hal penginputan data, upload image, dan print report dan penambahan graph statistik di dashboard
